# aqieFrame
- core : 框架核心类文件
- databases:数据库驱动目录
- helpers: 辅助函数目录
- lbraries: 类库目录，工具类模型


 ## 统一命名规范
 - （文件名 ：
 -  类文件 ：AqieController.class.php
 -  控制器 ：AqieController
 -  模型   ：AqieModel
 -  方法名 ：aqieAction
 -  属性名 : 小驼峰或者下划线
 -  函数名 ：下划线var_dump imagecreatetruecolor
 -  常量名 : 大写
